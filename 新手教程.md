# 第一个份试卷

1. 安装node.js,如果已安装可以跳过

```
npm install  nodejs -g
```

2. 编写题目文件，保存成`data.txt`

```
<H>初中地理考试
<P>一、判断题
<?=y>1.中国的陆地面积是960万平方千米？
<P>二、填空题
<.=1949>1.中华人民共和国成立于{$}年
<P>三、单选题
<*=D>1.下面是哪个不是海?
A.日本海
B.南海
C.里海
D.死海
<P>四、多选题
<**=AC>1.下面哪几个是首都?
A.伦敦
B.纽约
C.新加坡
D.仁川
<P>五、问答题
<_=因为地球公转一周要365天>2、为什么一年有365天？
```

3. 生成，运行如下命令
```
node run.js data.txt
```
