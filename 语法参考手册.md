# 语法

### 判断题

语法: <?=y>,=y 表示答案是对，可选

示例：

```
<?>1.中国的陆地面积是960万平方千米？
<?=y>1.中国的陆地面积是960万平方千米？
<?=n>2.因为中国使用的是东八区时间，所以中国在东八区？
```


### 问答题

语法: <_=中国>，=中国 表示答案是“中国”，可选

示例：

```
<_>2、为什么一年有365天？
<_=因为地球公转一周要365天>2、为什么一年有365天？
```

### 填空题

语法: <.=中国>，=中国 表示答案是“中国”，可选；{$} 表示空出现的位置

示例：

```
<.>1.中华人民共和国成立于{$}年
<.=1949>1.中华人民共和国成立于{$}年
```

### 单选题

语法: <*=A>，=A 表示答案是“A”，可选；一行表示一个选项

示例：

```
<*=D>1.下面是哪个不是海?
A.日本海
B.南海
C.里海
D.死海
```
### 多选题

语法: <**=A>,=AB 表示答案是“AB”，可选；一行表示一个选项

示例：

```
<**=AC>1.下面哪几个是首都?
A.伦敦
B.纽约
C.新加坡
D.仁川
```

### 标题

语法: <T>小学语文六年级考试

示例：

```
<T>小学语文六年级考试
```

### 大题标题

语法：<P> 一、选择题

示例：

```
<P> 一、选择题
```
